# Nginx Shield Lite

零依赖单 Python 文件 Web 应用，集成 Nginx 日志分析、爬虫识别、IP 黑名单管理功能。

![截图](preview.jpg)

## 功能

- **爬虫统计** — 识别 9 大搜索引擎爬虫，支持仅当天、仅成功请求筛选
- **IP 访问统计** — 去重统计每个 IP 访问次数，可调阈值、按访问量或 IP 地址排序
- **多级封禁** — 支持单 IP、C 段（/24）、B 段（/16）、A 段（/8）一键封禁
- **黑名单编辑** — 在线编辑 Nginx `geo` 黑名单，支持重复检测、IP 排序、一键保存
- **Nginx 操作** — 浏览器内检查配置和重载 Nginx
- **封禁状态** — 已封禁 IP 自动标红显示，支持按 CIDR 段匹配

## 零依赖

仅使用 Python 3 标准库，无需任何框架，直接运行，开箱即用。

## 快速开始

### 1. 配置 Nginx

在 `nginx.conf` 的 `http` 块中添加 geo 规则：

```nginx
http {
    geo $blacklisted_ip {
        default 0;
        include conf.d/black_ip.conf;
    }
}
```

在每个需要防护的 `server` 块中添加拦截：

```nginx
server {
    if ($blacklisted_ip) {
        return 444;
    }
    # ...
}
```

重载 Nginx 使配置生效：`nginx -s reload`

### 2. 启动

修改 `main.py` 顶部配置后运行：

```bash
bash restart.sh
```

浏览器访问 `http://你的服务器IP:9999` 即可。

## 使用指南

### 日志分析

- **爬虫统计** — 勾选「只看当天」或「只看成功请求」可筛选数据
- **IP 统计** — 通过 ±按钮 调整访问次数阈值，按访问量或 IP 地址排序
- **只看未封禁** — 勾选后仅显示未被黑名单覆盖的 IP
- 已封禁的 IP 会显示为红色

### 封禁 IP

1. 在 IP 统计表中点击封禁按钮：**单IP** / **C段** / **B段** / **A段**
   - 单 IP：仅屏蔽该 IP
   - C 段：屏蔽 /24 网段（256 个 IP）
   - B 段：屏蔽 /16 网段（65536 个 IP）
   - A 段：屏蔽 /8 网段（约 1677 万个 IP，点击后需二次确认）
2. 在右侧「IP 黑名单编辑」中点击 **检查重复**
3. 点击 **保存**
4. 点击 **检查配置**，确认无误后点击 **重载Nginx** 生效

### 黑名单文件格式

`black_ip.conf` 中每行一条规则：

```
8.228.46.180 1;
14.29.109.0/24 1;
14.116.236.0/24 1;
```

- 单个 IP：`8.228.46.180 1;`
- IP 段：`14.29.109.0/24 1;`
- 末尾 `1` 表示命中（对应 `geo` 中 `default 0`）

## 工作原理

```
Nginx access.log
       │
       ▼
  ┌──────────┐   HTTP API    ┌──────────┐
  │  main.py │◄──────────────│ browser  │
  │ (port    │──────────────►│   UI     │
  │  9999)   │   JSON data   │          │
  └──────────┘               └──────────┘
       │
       ├── 读写 ──────────► black_ip.conf
       └── subprocess ────► nginx -t / nginx -s reload
```
